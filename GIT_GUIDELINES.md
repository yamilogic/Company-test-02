# Git Guidelines

- **Branches:** `[initials]/[platform]/[type]/[description]` (e.g. `ya/fe/feat/mobile-menu`)
- **Commits:** `<type>(<platform>): <description>` (e.g. `feat(fe): add mobile menu`)
- **Initials:** `ya` (Yami), `ac` (Amit Kumar), `cr` (Creator)
- **Platforms:** `fe` (Frontend), `be` (Backend), `ops` (DevOps/CI-CD)
- **Types:** `feat` (features), `fix` (bugfixes), `refact` (refactoring), `docs` (docs)
